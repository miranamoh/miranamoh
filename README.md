## About Me

I'm an Artificial Intelligence undergraduate with a strong interest in building Machine Learning and Deep Learning systems that go beyond the notebook — practical AND well-engineered. What draws me to AI is the process of turning raw, messy data into something that actually works and solves a real problem: a model that predicts, a system that recognizes, a dashboard someone can interact with.

My work so far spans three areas I'm particularly passionate about:

- **Time-series forecasting** — modeling real-world economic indicators, including a project forecasting Saudi Arabia's unemployment rate using statistical and machine learning approaches
- **Computer vision** — building systems that detect and recognize objects in real time, from product recognition to image classification with convolutional neural networks
- **Applied deep learning** — using transfer learning and fine-tuning strategies to get strong results efficiently, rather than training large models from scratch

I care as much about *how* AI project is built as *what* it does — clean, reproducible code, thoughtful evaluation, and documentation that makes a project easy for someone else to pick up and understand.
- Currently strengthening my ML/CV project portfolio for AI/ML  applications
-  Currently deepening my knowledge of computer vision (YOLO, CLIP) and MLOps fundamentals
